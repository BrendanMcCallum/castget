castget

A command-line podcast downloader

castget is a simple, command-line based RSS enclosure downloader. It is primarily intended for automatic, unattended downloading of podcasts.

castget was written and is maintained by Marius L. JÃ¸hndal.

Features

  • Simple configuration.
  • Per-channel ID3 tagging of downloaded MP3 files.
  • Per-channel download directories.

Download

There is currently no stable version of castget. You may, however, download the latest development tarball http://savannah.nongnu.org/download/castget/castget-0.9.0.tar.gz.

To build castget you will need

  • glib2
  • libxml2
  • libcurl

and optionally

  • id3lib

castget has only been tested on Linux, but should work on other UNIX-like systems.

Configuration

castget is configured using a configuration file called .castgetrc in the user's home directory. The source distribution includes a sample configuration file.

Development

Development of castget takes place on Savannah.
